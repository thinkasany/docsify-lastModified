# 自我介绍
docsify插件: 最后一次编辑时间

使用`Minify`压缩

# [npm](https://www.npmjs.com/package/docsify-lastmodified)


# 🍔配置
```
repo: 'doocs/advanced-java'
lastModifiedText: '最近更新时间：'
```
